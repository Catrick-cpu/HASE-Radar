## WHAT THIS PROJECT IS

A real hardware radar system, not a simulation. Someone is building a 10 GHz phased-array FMCW radar to:
- Demonstrate radar technology in the 10 GHz amateur radio band
- Operate legally in Germany under a Klasse A amateur license
- Stay under $8,000 USD budget
- Be fully open-source and reproducible

The design is **complete at the engineering level** — all documentation, schematics, code, and STL files exist. What does NOT exist yet: physical hardware, verified EM simulations, tested firmware.

6. Layout rules from `PCB/Manufacturing_Notes.md`:

   - RF traces: use curves not right angles
   - Ground: solid copper pour on layer 2
7. Order from PCBWay (Rogers boards) or JLCPCB (FR4 boards)

### "Add feature X to the control software"
- File: `Software/Control/aeris_control.py`
- Protocol: USB CDC-ACM, packet format in main.c docstring
- To add a new command: (1) assign command byte ID (increment from last), (2) add case in `Process_USB_Command()` in main.c, (3) add method in `AERISController` class in aeris_control.py

- Test: `python aeris_control.py --help` for all current commands

### "Verify antenna design / run simulation"
1. Run: `python RF_System/openems_antenna_simulation.py --mode analytical`

   - Should output: W=7.92mm, L=13.8mm, resonance ≈10.05 GHz
2. For full EM sim: install OpenEMS (`conda install -c conda-forge openems`)
3. Run: `python openems_antenna_simulation.py --mode setup --sim-path ./sim`
4. Execute: `bash ./sim/run_simulation.sh`
5. Post-process: `python openems_antenna_simulation.py --mode postprocess --sim-path ./sim`
6. Pass criteria: S11 < -15 dB at 10.00–10.10 GHz

## TECHNICAL REFERENCE (QUICK LOOKUP)

### ADF4159 Chirp Register Summary (10.000–10.100 GHz, 1 ms)
```
f_ref  = 10.000 MHz (OCXO)
f_pfd  = 10.000 MHz (R=1)
f_start = 10.000 GHz → INT=1000, FRAC=0
f_stop  = 10.100 GHz → ramp via DEV_WORD
CLK_DIV = 4 (CLK2 = 2.5 MHz)
n_steps = 2500 (in 1 ms at CLK2 rate)
delta_f = 100 MHz / 2500 = 40 kHz/step
DEV_WORD = 40000 * 4096 / 10000000 = 16 (→ actual step 39.0625 kHz)
R0 = 0xCA000000 | (INT << 15) | (FRAC_MSB << 3) = 0xCA7D0000
R7 = 0x00000007 (ramp disabled initially)
```

### Phase Shifter SPI Protocol
```
SPI1 (TX) or SPI2 (RX): CPOL=0, CPHA=0, 10 MHz, 8-bit
Transaction: 16 bytes (one per shifter)
Order: Byte[0] → last element in chain (PSH_TX[15])
       Byte[15] → first element in chain (PSH_TX[0])
CS: active LOW, hold LOW for all 16 bytes, pulse HIGH to latch
Phase code: 6-bit, [5:0] = [180°:90°:45°:22.5°:11.25°:5.625°]
Example: 90° = 0b010000 = 0x10 = 16 decimal
```

### Beam Steering Formula
```
Phase for element at position (row, col) for steering (az_deg, el_deg):
  phi_deg = 180 * ((row-1.5)*sin(az_rad) + (col-1.5)*sin(el_rad))
  normalize to [0, 360)
  code = round(phi_deg / 5.625) % 64
Example: steer 30° az, element (3,0):
  phi = 180 * ((3-1.5)*sin(30°) + (0-1.5)*sin(0°)) = 180 * 1.5*0.5 = 135°
  code = round(135/5.625) = 24 = 0b011000
```

### Link Budget at 3 km (Car Target)
```
TX power:          +27.0 dBm (PA output, conservative)
TX cable loss:      -0.5 dB
TX divider loss:   -13.0 dB
TX phase shift:     -4.0 dB
TX feed:            -0.3 dB
TX array gain:     +22.0 dBi (16 elements × efficiency)
= TX EIRP:         +31.2 dBm

Free-space (2-way at 3km, 10GHz): -146.8 dB
Target RCS (car, 10m²):           +10.0 dBsm

RX array gain:     +22.0 dBi
RX phase shift:     -4.0 dB
RX combiner:        -1.0 dB
= Signal at mixer: -88.6 dBm

System NF:          ~2.0 dB
Noise BW (IF):      10 Hz (for 100 chirps × 1ms)
Noise power:       -172 + 10 + 2 = -160 dBm (approx)

SNR = -88.6 - (-160) = +71 dB  [theoretical maximum]
After range FFT + Doppler FFT processing losses: ~+40 dB practical SNR
Detection threshold: 15 dB → 25+ dB margin → WELL DETECTED at 3 km
```
